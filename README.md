# Bash utilities

# Instructions

## Clone Repository

```bash
git clone https://github.com/MiguelYax/bash-utilities.git ~/dev/github/bash-utilities

```

### Add bash-utilities config to your shell file configuration

- Add execution permisions
```bash
chmod +x install.sh
```

- Add to .bashrc
```bash
 ./install.sh ~/.bashrc
```

- Add to .zshrc
```bash
 ./install.sh ~/.zshrc
```
